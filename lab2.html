2. Accessibility

Provide Alternative Text for Images (WCAG Guideline 1.1.1): Ensure that all images on the website have descriptive alt text. This helps users with visual impairments understand the content of images when screen readers are used. Implement Keyboard Navigation (WCAG Guideline 2.1.1): Ensure that all interactive elements, such as buttons and links, can be easily navigated and activated using a keyboard alone, without relying on a mouse or touch input. This is for users with mobility impairments.
